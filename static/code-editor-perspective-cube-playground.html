Reference & Instructions

Key Parameters to Adjust:

  • perspective: Lower values (500-800px) = stronger effect, Higher values (1000-1500px) = subtler effect
  • rotateY: Negative values rotate left, positive rotate right (-45deg to 45deg)
  • rotateX: Negative values tilt up, positive tilt down (-25deg to 25deg)
  • perspective-origin: Controls the vanishing point (50% 50% = center, adjust Y for angle)
  • translateZ: Depth of side faces (half width/height of element)
