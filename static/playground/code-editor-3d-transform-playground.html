SVG Textarea 3D Transform Playground

Option 1: Double Stroke Layers (Teal/Gray)
Option 2: Glowing Stroke Layers (Orange/Pink)
Option 3: Dashed Stroke Layers (Blue)
Option 4: Multi-color Stroke Layers (Orange + Blue)
Reference: Hover over each SVG textarea to see the transform effect

This playground tests 3D transform effects on SVG textarea with different stroke-based depth layers. Each option uses ::before and ::after pseudo-elements with borders/strokes instead of transparent fills. Hover over each textarea to see it scale up and the stroke layers adjust.
